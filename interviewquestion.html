SQL Interview Questions

1. What is SQL?

SQL (Structured Query Language) is a standard language for accessing and manipulating databases.

2. What is the difference between DELETE and TRUNCATE?

- DELETE: Removes specific rows from a table and can have a WHERE clause. Logs individual row deletions.
- TRUNCATE: Removes all rows from a table and resets identity. Faster but not logged row by row.

3. What is a Primary Key?

A Primary Key uniquely identifies each record in a table. It must have unique values and cannot be NULL.

4. Explain JOINs in SQL.

JOINs are used to combine rows from two or more tables:

  • INNER JOIN: Returns records with matching values in both tables.
  • LEFT JOIN: Returns all records from the left table and matching records from the right table.
  • RIGHT JOIN: Returns all records from the right table and matching records from the left table.
  • FULL JOIN: Returns all records when there is a match in either table.

5. How do you retrieve duplicate records from a table?


SELECT column_name, COUNT(*)
FROM table_name
GROUP BY column_name
HAVING COUNT(*) > 1;

6. What is a Foreign Key?

A Foreign Key is a column that creates a relationship between two tables by referencing the Primary Key of another table.

7. What are ACID properties?

ACID stands for:

  • Atomicity: Transactions are all-or-nothing.
  • Consistency: Ensures data remains consistent before and after transactions.
  • Isolation: Transactions do not affect each other.
  • Durability: Once a transaction is committed, it remains permanent.

8. How do you find the nth highest salary in a table?


SELECT DISTINCT salary
FROM employees
ORDER BY salary DESC
LIMIT n-1, 1;

9. What is the difference between WHERE and HAVING?

- WHERE: Filters rows before grouping.
- HAVING: Filters groups after aggregation.

10. What are indexes in SQL?

Indexes are used to speed up data retrieval from a table by creating pointers to the data.

11. How can you improve query performance?

  • Use indexes on frequently searched columns.
  • Avoid SELECT *; only fetch required columns.
  • Use JOINS instead of subqueries where possible.
  • Analyze query execution plans to find bottlenecks.

12. Explain the difference between UNION and UNION ALL.

- UNION: Combines results from two queries and removes duplicates.
- UNION ALL: Combines results but includes duplicates.

13. How do you create a stored procedure?


CREATE PROCEDURE procedure_name AS
BEGIN
  -- SQL statements
END;

14. What is the use of the GROUP BY clause?

The GROUP BY clause groups rows that have the same values into summary rows, often used with aggregate functions like COUNT, SUM, AVG, etc.

15. What are triggers in SQL?

Triggers are special procedures that automatically execute in response to certain events on a table (INSERT, UPDATE, DELETE).

16. What is a view in SQL?

A view is a virtual table based on the result of a SELECT query.

17. How do you handle NULL values in SQL?

Use functions like IS NULL, IS NOT NULL, and COALESCE to manage NULL values.

18. Explain the difference between DROP, DELETE, and TRUNCATE.

  • DROP: Deletes the table and its structure.
  • DELETE: Deletes specific rows but retains the table structure.
  • TRUNCATE: Deletes all rows and resets table identity.

19. How do you add a new column to an existing table?


ALTER TABLE table_name ADD column_name datatype;

20. What is a cursor in SQL?

A cursor is a database object used to retrieve, manipulate, and navigate through a result set row by row.
