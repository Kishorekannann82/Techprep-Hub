Introduction to SQL

SQL (Structured Query Language) is used to interact with databases. It is the standard language for querying and manipulating relational databases. SQL is supported by many database systems such as MySQL, PostgreSQL, SQLite, Oracle, and Microsoft SQL Server.

SQL Basics

  • Creating Tables: CREATE TABLE table_name (column_name datatype, ...)
  • Inserting Data: INSERT INTO table_name (columns) VALUES (values)
  • Selecting Data: SELECT * FROM table_name
  • Updating Data: UPDATE table_name SET column=value WHERE condition
  • Deleting Data: DELETE FROM table_name WHERE condition

Common SQL Data Types

  • INT: Integer values
  • VARCHAR: Variable-length string
  • TEXT: Long text data
  • DATE: Date values (YYYY-MM-DD)
  • DECIMAL: Precise numeric values (e.g., 10.2 for 2 decimal places)
  • BOOLEAN: True/False values

Key SQL Clauses

  • WHERE: Filters rows based on a condition. Example: SELECT * FROM employees WHERE salary > 50000;
  • GROUP BY: Groups rows sharing a property. Example: SELECT department, COUNT(*) FROM employees GROUP BY department;
  • HAVING: Filters groups after aggregation. Example: SELECT department, COUNT(*) FROM employees GROUP BY department HAVING COUNT(*) > 5;
  • ORDER BY: Sorts results. Example: SELECT * FROM employees ORDER BY salary DESC;
  • LIMIT: Restricts the number of rows returned. Example: SELECT * FROM employees LIMIT 10;

SQL Joins

Joins are used to combine rows from two or more tables based on a related column.

  • INNER JOIN: Returns matching rows from both tables.
  • LEFT JOIN: Returns all rows from the left table, and matching rows from the right table.
  • RIGHT JOIN: Returns all rows from the right table, and matching rows from the left table.
  • FULL JOIN: Returns rows when there is a match in either table.

SELECT employees.name, departments.name
FROM employees
INNER JOIN departments
ON employees.department_id = departments.id;

SQL Functions

SQL includes built-in functions for performing calculations on data:

  • Aggregate Functions:
    • COUNT(): Returns the number of rows.
    • SUM(): Calculates the sum of a column.
    • AVG(): Calculates the average value of a column.
    • MAX() and MIN(): Returns the maximum or minimum value.
  • String Functions:
    • CONCAT(): Concatenates strings. Example: SELECT CONCAT(first_name, ' ', last_name) AS full_name FROM employees;
    • UPPER() and LOWER(): Converts strings to upper/lower case.
    • SUBSTRING(): Extracts a substring from a string.

SQL Interview Questions

  1. What is the difference between DELETE and TRUNCATE?

    - DELETE: Removes specific rows and can have a WHERE clause. It is logged.
    - TRUNCATE: Removes all rows without logging individual row deletions.

  2. What is a Foreign Key?

    A Foreign Key is a column that establishes a relationship between two tables.

  3. How to find the nth highest salary in a table? 
    
SELECT DISTINCT salary 
FROM employees 
ORDER BY salary DESC 
LIMIT 1 OFFSET (n-1);
  4. What are SQL Indexes?

    Indexes are used to speed up the retrieval of rows by creating pointers to data in a table. They improve query performance but require additional storage space.

  5. What is the difference between UNION and UNION ALL?

    - UNION: Combines result sets and removes duplicates.
    - UNION ALL: Combines result sets without removing duplicates.
